# Linux‑Upscaler

Skeleton project for a Linux‑native window/desktop upscaler (X11/Wayland + OpenGL/Vulkan).

## Structure

- external/        → external dependencies (e.g. xcap)  
- src/             → source code (capture, renderer, UI, utils)  
- shaders/         → GLSL / SPIR‑V shaders (upscaling/filter pipelines)  
- CMakeLists.txt   → build configuration  
- README.md        → project description
